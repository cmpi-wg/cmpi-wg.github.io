CMPI API documentation

This site provides the API documentation for the CMPI standard, starting with CMPI 2.1.

CMPI stands for "Common Manageability Programming Interface" and is a C-language API between a Management Broker (aka CIMOM) and Management Instrumentation (aka Providers). CMPI supports the CIM and WBEM standards defined by DMTF.

The CMPI Standard is maintained on the CMPI Working Group web space at OpenGroup; the CMPI header files for the CMPI Standard are maintained in the cmpi-wg/cmpi-headers project on Github.
