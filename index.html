CMPI Headers

This site provides the CMPI header files and their API documentation, as well as links to the underlying CMPI Standard documents.

CMPI stands for "Common Manageability Programming Interface" and is a C-language API between a Management Broker (aka CIMOM) and Management Instrumentation (aka Providers). CMPI supports the CIM and WBEM standards defined by DMTF. The CMPI Standard is maintained on the CMPI Working Group web space of the Open Group. The CMPI header files for the CMPI Standard are maintained in the cmpi-wg/cmpi-headers project on GitHub.
