What this site is about

This site provides the CMPI Standard, the CMPI header files, and the CMPI API documentation.

CMPI stands for "Common Manageability Programming Interface" and is a C-language API between a Management Broker (aka CIMOM) and Management Instrumentation (aka Providers). CMPI supports the CIM and WBEM standards defined by DMTF.

CMPI 2.1.0

CMPI 2.1.0 refers to CMPI Issue 2.1 without any Corrigenda.

Status as of 2015-10-01): Finalized by the CMPI WG and in the Open Group release process.
