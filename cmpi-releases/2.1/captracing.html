Tracing capability

Entities that are part of the Tracing capability

This capability is optional to be supported.

Support for this capability can be detected from CMPIBrokerFT.brokerCapabilities using the CMPI_MB_Tracing test mask.

If this capability is supported, the entities in the following list shall be supported; otherwise, they shall not be supported.
