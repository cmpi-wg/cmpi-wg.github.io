Older Versions

This page lists the older published versions of the CMPI Standard. The latest version is listed on the home page.

CMPI 2.0.0

CMPI 2.0.0 refers to CMPI Issue 2.0 without any Corrigenda.

Status: Published.

  • CMPI 2.0 header files
  • CMPI 2.0 API documentation in HTML (online)
  • CMPI Issue 2.0, December 2006, Document C061

CMPI 1.0.2

CMPI 1.0.2 refers to CMPI Issue 1.0 with Corrigenda 1 and 2.

Status: Published.

  • CMPI Issue 1.0, December 2004, Document C051
  • Corrigendum 1, October 2005, Document U064
  • Corrigendum 2, August 2006, Document U066
